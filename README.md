# CodeAlpha Python Programming Internship Tasks

This repository contains the practical Python projects assigned and completed during my internship at CodeAlpha. Each project focuses on core programming logic, automation, and user interaction.

## 📁 Projects Overview

### 1. Hangman Game (`/Task-1-Hangman_Game`)
* **Description:** A classic terminal-based word-guessing game featuring dynamic hidden-word updates, turn tracking, input validation, and random word selection using the `random` module.

### 2. Stock Portfolio Tracker (`/Task-2-Stock_Portfolio_Tracker`)
* **Description:** A console application that allows users to add or update stock assets, tracks real-time hardcoded pricing valuations, displays a structured financial report, and exports the data to a `.txt` file.

### 3. File Organizer Automation (`/Task-3-Task_Automation_Python_Scripts`)
* **Description:** A utility script using `os` and `shutil` modules to scan directories, filter specific file formats (`.jpg`), organize them into destination folders, and generate an automated execution report.

### 4. Basic Chatbot (`/Task-4-Basic_Chatbot`)
* **Description:** A simple, rule-based conversational script that interacts with users based on predefined keywords (greetings, status checks, and exit commands).


## 🛠️ Technologies Used
* Python 3.x
* Built-in libraries: `os`, `shutil`, `random`
